# task\<br\>eaker
simple CLI task manager written in Go.

## about
task\<br\>eaker allows you to add your tasks. mark them as done, save them in JSON.\
this is my pet-project in MVP stage.\
i am happy to share it even though it is hardly anything new.

## features
- add new
- mark as done
- list all
- store in JSON
- exit & save

## installation
clone the repository and build the project:
```
git clone https://github.com/temaelkin/taskbreaker.git
cd taskbreaker
go build
```

## usage
1. run the application.
2. use `help` for available commands

## motivation
this project was created as a learning exercise to practice Go fundamentals,\
project structure, separation of concerns and basic CLI design.\
